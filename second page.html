Important CSS Code

Here are some CSS styles you should know:

/* Change text color and size */
p {
  color: blue;
  font-size: 16px;
}
/* Add background color */
body {
  background-color: #f0f0f0;
}
/* Set margins and padding */
div {
  margin: 20px;
  padding: 10px;
}
/* Add a border */
.box {
  border: 2px solid black;
  border-radius: 10px;
}
/* Center text */
h1 {
  text-align: center;
}
/* Add a hover effect */
button:hover {
  background-color: lightblue;
}
