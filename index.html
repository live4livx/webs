Important HTML Tags

Here are some basic HTML tags everyone should know:

  • <html> - The root of the HTML document.
  • <head> - Contains meta information about the page.
  • <title> - Sets the title of the page (seen on the tab).
  • <body> - Contains everything you see on the page.
  • <h1> to <h6> - Headings (h1 is the biggest).
  • <p> - Paragraphs of text.
  • <a> - Links to other pages.
  • <img> - Adds images.
  • <ul>, <ol>, <li> - Lists (unordered and ordered).
  • <div> - A container for other content.
  • <span> - Used to style small parts of text.
  • <br> - Line break.
